# 🚀 Uplift - App de Frases Motivacionais

App Android profissional de frases motivacionais com Firebase e monetização via AdMob.

## ✨ O que é o Uplift?

Uplift é um aplicativo completo que oferece:
- 📖 Frases motivacionais por categoria (Motivação, Sucesso, Amor, Vida, Sabedoria)
- ⭐ Sistema de favoritos
- 🔄 Compartilhamento social (WhatsApp, Instagram, etc)
- 🔔 Notificações diárias
- 💰 Monetização completa com AdMob (banners + intersticiais)
- ☁️ Firebase Firestore (database em nuvem)

## 🎯 Funcionalidades

✅ Interface moderna com Material Design
✅ Lista infinita de frases com RecyclerView
✅ Sistema de categorias
✅ Favoritos salvos localmente
✅ Compartilhamento direto para redes sociais
✅ Notificações diárias automáticas
✅ Anúncios estratégicos (a cada 5 ações)
✅ Database em nuvem com Firebase

## 💰 Monetização

Para monetizar, você precisa:
1. Configurar Firebase (obrigatório)
2. Criar conta no AdMob
3. Substituir IDs de teste por IDs reais
4. Publicar na Google Play Store

**Potencial de Receita:**
- 1.000 usuários/dia: $10-30/mês
- 5.000 usuários/dia: $50-150/mês
- 10.000 usuários/dia: $100-300/mês
